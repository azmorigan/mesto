### О проекте:
Одностраничный адаптивный сайт с возможностью менять имя и деятельность в профиле при помощи формы, всплывающей при нажатии на кнопку редактирования.

Использованные технологии и ПО:
* HTML, CSS, JavaScript
* Figma, VS Code
* BEM, Flex, Grid, DOM
